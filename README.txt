Polski

Prosta aplikacja webowa stworzona w ramach nauki języka PHP, obsługi sesji oraz obsługi bazy danych MySQL, jako projekt akademicki z przedmiotu "Techniki Internetu". W projekcie zostały wykorzystane technologie: PHP, HTML, SQL i CSS.

Składa się ona z następujących części:
-strony głównej sklepu
-strony koszyka
-strony formularza

Użytkownik może:
-dodawać oraz usuwać przedmioty z koszyka
-otrzymać maila z potwierdzeniem zakupów po wypełnieniu formularza

Aplikacja wykorzystuje sesję do przechowywania wprowadzonych danych. Połączenie z bazą danych MySQL umożliwia pobieranie danych.
Do poprawnego funkcjonowania aplikacji niezbędna jest poprawna konfiguracja plików sendmail.ini oraz php.ini.

English

A simple web application created as part of learning PHP language, session management, and MySQL database management, as an academic project for the "Internet Techniques" subject. Technologies used in the project include: PHP, HTML, SQL and CSS.

It consists of the following parts:
-main shop page
-cart page
-form page

The user can:
-add and remove items from the cart
-receive an email confirmation of purchases after filling out the form

The application uses a session to store entered data. The connection to the MySQL database allows for data retrieval.
Correct configuration of sendmail.ini and php.ini files is necessary for the proper functioning of the application.
